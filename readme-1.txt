Program:
Untuk nomor 1 uts grafkom kami membuat game bernama basket catcher.
Program ini membuat pemain menggerakkan "basket" untuk menangkap balok-balok
yang jatuh dari atas. Balok merah bernilai -5 poin, balok biru bernilai 5 poin
dan balok putih berniali 25 poin. Tujuan pemain adalah mendapatkan 500 poin sebelum 
waktu habis. Untuk menggeraka basket pemain memencet Left/Right arrow key dan untuk
memulai gamenya pemain memencet tombok SPACE pada keyboard.

Struktur:
file yang digunakan adalah index.html, player.js dan block.js dan juga common
